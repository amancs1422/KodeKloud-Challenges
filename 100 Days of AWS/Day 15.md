# Day 15: Create volume from Snapshot
Create a snapshot of an existing volume named devops-vol in us-east-1 region.
1) The name of the snapshot must be devops-vol-ss.
2) The description must be devops Snapshot.
3) Make sure the snapshot status is completed before submitting the task.
## Step 1 : Find the "Volumes" section in EC2 dashboard of AWS Management Console.
![](https://github.com/amancs1422/KodeKloud-Challenges/blob/d501a6f39f1a6474321fcb83c1b7b3dd82ce7851/100%20Days%20of%20AWS/Images/Day%2015_1.jpg)
## Step 2 : Select the "devops-vol" and click on "Create Snapshot" button as shown below.
![](https://github.com/amancs1422/KodeKloud-Challenges/blob/d501a6f39f1a6474321fcb83c1b7b3dd82ce7851/100%20Days%20of%20AWS/Images/Day%2015_2.jpg)
## Step 3 : Provide the description of the Snapshot as per lab requirement : "devops Snapshot" and the finally click on "Create Snapshot" button.
![](https://github.com/amancs1422/KodeKloud-Challenges/blob/d501a6f39f1a6474321fcb83c1b7b3dd82ce7851/100%20Days%20of%20AWS/Images/Day%2015_3.jpg)
